# Project Planner - Design System

Terminal-inspired aesthetic: sharp edges, flat colors, monospace typography. No gradients, no rounded corners.

---

## Colors

### Backgrounds

| Token | Hex | Usage |
|-------|-----|-------|
| `bg-base` | `#0a0a0a` | Page background |
| `bg-surface` | `#141414` | Cards, panels |
| `bg-elevated` | `#1f1f1f` | Modals, dropdowns |
| `bg-muted` | `#262626` | Hover states, subtle emphasis |

### Text

| Token | Hex | Usage |
|-------|-----|-------|
| `text-primary` | `#fafafa` | Headings, important content |
| `text-secondary` | `#a1a1a1` | Body text, descriptions |
| `text-muted` | `#737373` | Placeholders, disabled |

### Accent

| Token | Hex | Usage |
|-------|-----|-------|
| `accent` | `#22c55e` | Primary actions, links, focus |
| `accent-hover` | `#16a34a` | Hover state |

### Status

| Token | Hex | Usage |
|-------|-----|-------|
| `success` | `#22c55e` | Completed, positive |
| `warning` | `#eab308` | Deadlines, caution |
| `error` | `#ef4444` | Overdue, destructive |
| `info` | `#3b82f6` | In progress, informational |

### Priority

| Token | Hex | Usage |
|-------|-----|-------|
| `priority-high` | `#ef4444` | High priority indicator |
| `priority-medium` | `#eab308` | Medium priority indicator |
| `priority-low` | `#3b82f6` | Low priority indicator |

## Typography

### Font Families

| Token | Value | Usage |
|-------|-------|-------|
| `font-mono` | `'JetBrains Mono', 'Fira Code', monospace` | UI elements, code |
| `font-sans` | `system-ui, sans-serif` | Long-form text (optional) |

### Font Sizes

| Token | Size | Line Height |
|-------|------|-------------|
| `text-xs` | 12px | 16px |
| `text-sm` | 14px | 20px |
| `text-base` | 16px | 24px |
| `text-lg` | 18px | 28px |
| `text-xl` | 20px | 28px |
| `text-2xl` | 24px | 32px |

### Font Weights

| Token | Value | Usage |
|-------|-------|-------|
| `font-normal` | 400 | Body text |
| `font-medium` | 500 | Emphasis |
| `font-semibold` | 600 | Headings |

## Spacing

| Token | Value | Usage |
|-------|-------|-------|
| `space-1` | 4px | Tight gaps |
| `space-2` | 8px | Default gap |
| `space-3` | 12px | Related elements |
| `space-4` | 16px | Section padding |
| `space-6` | 24px | Card padding |
| `space-8` | 32px | Section margins |
| `space-12` | 48px | Large separations |
